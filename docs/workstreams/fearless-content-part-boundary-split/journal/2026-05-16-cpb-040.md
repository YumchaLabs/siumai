# CPB-040 Journal

Date: 2026-05-16

## Summary

Migrated one production response projection path away from direct legacy `ContentPart`
construction.

## Change

`siumai-core/src/streaming/processor.rs` now routes terminal text fallback projection through
`response_text_part(...)` instead of `ContentPart::text(...)`. The source guard prevents
`StreamProcessor` response consolidation from reintroducing direct text constructor calls, and a
behavior test verifies empty request provider options plus no response provider metadata for plain
terminal text.
