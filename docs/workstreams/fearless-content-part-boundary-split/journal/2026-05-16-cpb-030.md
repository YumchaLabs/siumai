# CPB-030 Journal

Date: 2026-05-16

## Summary

Migrated one production request construction path away from direct legacy `ContentPart`
construction.

## Change

`siumai-provider-google-vertex/src/standards/vertex_gemini_image.rs` now routes Gemini image prompt
text through `request_text_part(...)` instead of `ContentPart::text(...)`. The module's source guard
now covers both text and file request adapters, and a behavior test verifies that prompt text emits
empty request provider options and no response provider metadata.
