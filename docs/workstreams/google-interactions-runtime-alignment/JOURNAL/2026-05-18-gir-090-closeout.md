# GIR-090 Closeout

Date: 2026-05-18

## Decision

Close the Google Interactions runtime alignment lane without splitting a new Interactions-specific
follow-on.

## Rationale

The lane target state is covered:

- request conversion for model and agent modes;
- completed response parsing into stable `ChatResponse` content and metadata;
- non-stream `/interactions` execution and agent polling;
- model-mode SSE streaming;
- agent-mode background streaming through resumable `GET /interactions/{id}?stream=true`;
- `last_event_id` reconnect and best-effort cancel behavior;
- public facade parity across `Provider::google()`, `provider_ext::google`, and direct
  `GoogleInteractionsLanguageModel` construction.
